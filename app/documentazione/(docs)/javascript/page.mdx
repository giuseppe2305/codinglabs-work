# Javascript

## Cos'è JavaScript

JavaScript è un linguaggio di programmazione **interpretato**, **dinamico** e **orientato agli oggetti**, utilizzato principalmente per rendere le pagine web **interattive**.  
È uno dei tre pilastri fondamentali del web insieme a **HTML** e **CSS**.

- **HTML** → struttura della pagina
- **CSS** → stile e layout
- **JavaScript** → comportamento e interazione

## Dove viene usato

JavaScript può essere eseguito in diversi ambienti:

- **Browser** (Chrome, Firefox, Edge, ecc.)
- **Server** (tramite Node.js)
- **Applicazioni desktop** (Electron)
- **Applicazioni mobile** (React Native)
- **Backend e API**
- **Videogiochi e IoT**

## Caratteristiche principali

- Linguaggio **dinamicamente tipizzato**
- Supporta **programmazione funzionale** e **orientata agli oggetti**
- È **event-driven** (basato su eventi)
- Single-threaded con **event loop**
- Supporta operazioni **asincrone**

## Come funziona nel browser

Nel browser JavaScript:

- Risponde agli eventi (click, input, submit)
- Manipola il **DOM** (Document Object Model)
- Comunica con server tramite **HTTP / API**
- Aggiorna la pagina senza ricaricarla (AJAX)

## Sintassi di base

### Variabili

```javascript
let nome = "Mario";
const età = 30;
var città = "Roma";
```

### Tipi di dato

- String (`"testo"`)
- Number (`10`, `3.14`)
- Boolean (`true`, `false`)
- Object
- Array
- Null
- Undefined

### Funzioni

```javascript
function saluta(nome) {
  return "Ciao " + nome;
}
```

Oppure con arrow function:

```javascript
const saluta = (nome) => {
  return `Ciao ${nome}`;
};
```

## Strutture di controllo

### Condizioni

```javascript
if (età >= 18) {
  console.log("Maggiorenne");
} else {
  console.log("Minorenne");
}
```

### Cicli

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

## Oggetti e Array

### Oggetti

```javascript
const persona = {
  nome: "Luca",
  età: 25,
  saluta() {
    console.log("Ciao!");
  },
};
```

### Array

```javascript
const numeri = [1, 2, 3, 4];
numeri.push(5);
```

## Asincronicità

JavaScript gestisce operazioni asincrone tramite:

- Callback
- Promise
- Async / Await

```javascript
async function caricaDati() {
  const risposta = await fetch("https://api.example.com");
  const dati = await risposta.json();
  console.log(dati);
}
```

## Librerie e Framework

## Perché imparare JavaScript

- È uno dei linguaggi più richiesti
- Funziona su frontend e backend
- Grande ecosistema
- Comunità enorme
- Evoluzione costante

## Conclusione

JavaScript è un linguaggio estremamente versatile e potente, fondamentale per lo sviluppo web moderno. Impararlo significa aprire le porte a moltissime opportunità nel mondo della programmazione.
