# Introduzione a JavaScript

JavaScript è un linguaggio di programmazione **dinamico**, **interpretato** e **basato su oggetti**, utilizzato principalmente per sviluppare applicazioni web. Grazie a JavaScript è possibile rendere le pagine web interattive, manipolare il DOM (Document Object Model), gestire eventi, fare chiamate HTTP e molto altro.

È supportato da tutti i principali browser ed è uno dei linguaggi fondamentali del web, insieme a **HTML** e **CSS**.

---

## 1. Storia e caratteristiche principali

- Creato nel **1995** da Brendan Eich.
- Linguaggio **client-side** (anche se oggi esistono ambienti server-side come Node.js).
- Sintassi simile a C, con flessibilità nella tipizzazione.
- Basato su **oggetti**, ma supporta anche programmazione funzionale.
- Interpretabile direttamente dal browser senza necessità di compilazione.

### Caratteristiche principali

1. **Dinamico**: le variabili possono cambiare tipo durante l’esecuzione.
2. **Debolmente tipizzato**: il linguaggio converte automaticamente i tipi quando necessario.
3. **Event-driven**: ideale per gestire interazioni dell’utente.
4. **Cross-platform**: funziona su qualsiasi browser o server con ambiente JavaScript.

## 2. Sintassi di base

### 2.1 Variabili

In JavaScript possiamo dichiarare variabili con:

```javascript
var nome = "Mario"; // vecchio metodo
let eta = 25; // variabile moderna, modificabile
const PI = 3.14; // costante, non modificabile
```

- `var`: visibilità globale o funzione.
- `let`: visibilità di blocco.
- `const`: costante, visibile solo nel blocco in cui è dichiarata.

---

### 2.2 Tipi di dati

JavaScript supporta vari tipi di dati:

```javascript
// Tipi primitivi
let numero = 42; // Number
let testo = "Ciao"; // String
let booleano = true; // Boolean
let nullo = null; // Null
let indefinito; // Undefined
let simbolo = Symbol("id"); // Symbol
let bigInt = 12345678901234567890n; // BigInt

// Tipi complessi
let array = [1, 2, 3]; // Array
let oggetto = { nome: "Anna" }; // Object
```

---

### 2.3 Operatori

- **Aritmetici**: `+`, `-`, `*`, `/`, `%`, `**`
- **Assegnazione**: `=`, `+=`, `-=`, `*=`, `/=`
- **Confronto**: `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
- **Logici**: `&&`, `||`, `!`
- **Ternario**: `condizione ? valoreSeVero : valoreSeFalso`

## 3. Strutture di controllo

### 3.1 Condizioni

```javascript
let eta = 18;
if (eta >= 18) {
  console.log("Sei maggiorenne");
} else {
  console.log("Sei minorenne");
}

// switch
let giorno = 3;
switch (giorno) {
  case 1:
    console.log("Lunedì");
    break;
  case 2:
    console.log("Martedì");
    break;
  default:
    console.log("Altro giorno");
}
```

---

### 3.2 Cicli

```javascript
// while
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

// for
for (let j = 0; j < 5; j++) {
  console.log(j);
}

// for...of (array)
let array = ["a", "b", "c"];
for (let elemento of array) {
  console.log(elemento);
}

// for...in (oggetto)
let persona = { nome: "Anna", eta: 25 };
for (let chiave in persona) {
  console.log(chiave + ": " + persona[chiave]);
}
```

## 4. Funzioni

JavaScript supporta **funzioni tradizionali**, **funzioni anonime** e **arrow functions**:

```javascript
// Funzione tradizionale
function somma(a, b) {
  return a + b;
}

// Funzione anonima
let moltiplica = function (a, b) {
  return a * b;
};

// Arrow function
let sottrai = (a, b) => a - b;

console.log(somma(2, 3)); // 5
console.log(moltiplica(2, 3)); // 6
console.log(sottrai(5, 3)); // 2
```

---

## 5. Oggetti

Gli oggetti rappresentano **collezioni di proprietà** e possono avere metodi:

```javascript
let persona = {
  nome: "Luca",
  eta: 30,
  saluta: function () {
    console.log(`Ciao, sono ${this.nome}`);
  },
};

persona.saluta(); // Ciao, sono Luca
```

- `this` si riferisce all’oggetto corrente.
- Possiamo aggiungere proprietà dinamicamente:

```javascript
persona.citta = "Roma";
```

---

## 6. Array

Gli array contengono più valori ordinati:

```javascript
let numeri = [1, 2, 3, 4, 5];

// Metodi comuni
numeri.push(6); // aggiunge alla fine
numeri.pop(); // rimuove l'ultimo
numeri.shift(); // rimuove il primo
numeri.unshift(0); // aggiunge all'inizio

console.log(numeri.length); // lunghezza dell'array

// Iterazione con forEach
numeri.forEach((n) => console.log(n));
```

---

## 7. Eventi e DOM

JavaScript permette di interagire con il DOM:

```javascript
// Selezione di un elemento
let bottone = document.getElementById("mioBottone");

// Gestione eventi
bottone.addEventListener("click", () => {
  alert("Hai cliccato il bottone!");
});
```

- `getElementById`, `querySelector`, `querySelectorAll` sono metodi comuni per selezionare elementi.
- Possiamo modificare contenuti, attributi e stili dinamicamente.

---

## 8. Programmazione asincrona

JavaScript gestisce operazioni asincrone tramite:

### 8.1 Callback

```javascript
function fetchData(callback) {
  setTimeout(() => {
    callback("Dati ricevuti");
  }, 1000);
}

fetchData((data) => console.log(data));
```

### 8.2 Promises

```javascript
let promessa = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Successo"), 1000);
});

promessa.then((data) => console.log(data));
```

### 8.3 Async/Await

```javascript
async function fetchDati() {
  let risposta = await promessa;
  console.log(risposta);
}

fetchDati();
```

---

## 9. Moduli

Con ES6, JavaScript supporta moduli:

```javascript
// modulo.js
export const nome = "Mario";
export function saluta() {
  console.log("Ciao!");
}

// main.js
import { nome, saluta } from "./modulo.js";
console.log(nome);
saluta();
```
