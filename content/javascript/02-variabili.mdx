# Variabili in JavaScript

Le variabili servono a **memorizzare dati** che possono essere riutilizzati e manipolati nel codice.  
In JavaScript esistono **tre modi** per dichiarare una variabile: `var`, `let` e `const`. Non sono equivalenti.

---
