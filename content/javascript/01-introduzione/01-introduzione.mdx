# Introduzione a JavaScript

JavaScript è un linguaggio di programmazione **versatile e potente**, utilizzato principalmente per sviluppare applicazioni web. È il cuore dell'interattività dei siti web moderni e permette di creare esperienze dinamiche direttamente nel browser.

## Storia di JavaScript

JavaScript è stato creato nel **1995** da Brendan Eich e inizialmente era chiamato **Mocha**, poi **LiveScript**, fino ad arrivare al nome attuale. Oggi, JavaScript è supportato da **tutti i principali browser** e viene usato sia lato client che lato server grazie a piattaforme come **Node.js**.

## Perché imparare JavaScript

- Permette di creare siti web interattivi.
- È utilizzato nello sviluppo di **applicazioni web e mobile**.
- Ha una comunità enorme e risorse gratuite.
- È il linguaggio più richiesto nel mondo del lavoro per il web development.

## Sintassi di base

Ecco un esempio semplice di JavaScript che mostra un messaggio nel browser:

```javascript
console.log("Ciao, mondo!");
```

## Variabili

Le variabili sono contenitori per i dati. In JavaScript possiamo dichiararle con `var`, `let` o `const`:

```javascript
let nome = "Mario";
const eta = 20;
console.log(nome, eta);
```

## Funzioni

Le funzioni sono blocchi di codice riutilizzabili. Ecco un esempio:

```javascript
function saluta(nome) {
  return "Ciao " + nome + "!";
}

console.log(saluta("Luigi"));
```

## Conclusione

JavaScript è il linguaggio essenziale per chi vuole sviluppare sul web. Iniziare con concetti base come **variabili, funzioni e console** ti permetterà di costruire gradualmente applicazioni sempre più complesse.
